Gotham on Ouya

This is an XBMC Gotham build with Ouya specifics, i.e.

  • Ouya icon
  • Ouya passthrough
  • Reboot, ... on rooted devices

It is an "original" XBMC apk, with same signature, so it replaces any XBMC gotham/nightlies you might have.
