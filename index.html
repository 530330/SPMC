Semper Media Center (SPMC in short) is an unofficial fork of the official XBMC app.

Gotham

Those Gotham builds will contain XBMC patches which were not accepted, or are not acceptable, in the main XBMC tree.

Latest Gotham build is: 13.2.0 APK: https://github.com/koying/SPMC/releases/download/13.2.0-spmc/spmc-armeabi-v7a_13.2.0.apk

Changelog:

13.2.0:

  • Initial Gothan release
  • OUYA specifics
    • OUYA app specifics
    • AC3/DTS passthrough
  • Rockchip enhancements
    • Better H/W acceleration via private API
  • AML enhacements
    • 4K on M8

See https://github.com/koying/SPMC/commits/gotham for the full list.
